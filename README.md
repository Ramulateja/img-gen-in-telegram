# 🚀 Telegram AI Image Generator (n8n Automation)

This project is a simple automation built using n8n that allows you to generate AI images directly from Telegram.

Instead of switching between multiple tools, you can just send a prompt in Telegram and receive the generated image instantly.

---

## 🔥 What It Does

- Takes a text prompt from Telegram (e.g., "generate a dog image")
- Automatically triggers an n8n workflow
- Sends the prompt to an AI image generation model
- Returns the generated image back to Telegram

---

## ⚙️ How It Works

Telegram → n8n Workflow → AI Model → Telegram

### Flow:
1. User sends a message to Telegram bot  
2. n8n detects the message and triggers the workflow  
3. The prompt is sent to an AI image generator  
4. The generated image is sent back to the user  

---

AI tools like image generators already exist.

But this project focuses on:

This makes it faster and more practical for everyday use.

---

## 🛠️ Tech Stack

- n8n (workflow automation)
- Telegram Bot API
- AI Image Generation Model

---

## 📦 Setup

1. Import the `workflow.json` file into n8n  
2. Add your Telegram Bot Token  
3. Configure your AI API key  
4. Activate the workflow
